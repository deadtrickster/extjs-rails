# extjs-rails

This gem provides:

* ExtJS 4.2

Perfectly works with rails assets pipeline

# Installation
gem "extjs-rails"

gem "compass-rails"

``` js
//= require ext-all
```

``` css
//= require ext-all-dev
```


# Customization

``` ruby
config.extjs_theme
```
